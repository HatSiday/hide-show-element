HTML មកពីពាក្យថា hypertext markup language ហើយត្រូវបានគេប្រើប្រាស់ដើម្បីធ្វើការបង្កើត Web Page ដែលបង្ហាញនៅលើ​ Website ម្យ៉ាងវិញទៀត HTML ដូចទៅនិងគ្រោងឆ្អឹង ឬរចនាសម្ព័ន (structure) របស់ Web Page ។ នៅពេលដែល Web Page ច្រើនផ្គុំបញ្ចូលគ្នាបង្កើតបានជា Website ។
CSS មកពីពាក្យថា cascading style sheets គឺជា style sheet language មួយដែលត្រូវបាន web developer ប្រើប្រាស់ធ្វើការ រចនាបន្ថែមទៅលើ HTML Element ដើម្បីផ្លាស់ប្ដូរប្លង់ (Layout) និងរូបរាងរបស់គេហទំព័រ (web page)។ ឧទាហរណ៍៖ យើងប្រើ CSS ដើម្បីធ្វើការផ្លាស់ប្ដូរទំហំ ឬប្រភេទអក្សររបស់ HTML Element ណាមួយ។
JavaScript គឺជា client-side scripting language ដែលអនុញ្ញាតិឱ្យ developer អាចសរសេរកូដនៅក្នុង web page របស់ពួកគេ។ជាទូទៅកូដរបស់ JavaScript ត្រូវបានដាក់ចូលនៅក្នុង HTML(hypertext markup language) file ហើយដំណើរការដោយផ្ទាល់ពី web page ហើយសព្វថ្ងៃ JavaScript ក៏ជា most popular programming language មួយផងដែរ។
jQuery is a JavaScript Library.
